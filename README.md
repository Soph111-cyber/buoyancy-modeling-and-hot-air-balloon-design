# 🎈 Buoyancy Modeling & Hot-Air Balloon Design

This project explores the mathematical foundations of Archimedes’ principle using multivariable calculus and computational modeling.

## 🔬 Key Contributions

- Derived buoyancy force from first principles using double and triple integrals
- Interpreted buoyancy through geometric "column integration"
- Proved Archimedes’ principle using the divergence theorem
- Built an interactive simulator for hot-air balloon design

## 🧠 Core Insight

Buoyancy is not a separate force, but the global result of integrating a pressure field over a surface.
